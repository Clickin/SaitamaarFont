Saitamaar Font

A font designed specifically for ASCII art and Japanese SJIS-Art.

Webfont Usage

<link rel="stylesheet" href="https://YOUR-USERNAME.github.io/SaitamaarFont/webfonts/saitamaar.css">

CSS Usage

body {
font-family: 'Saitamaar', monospace;
}
